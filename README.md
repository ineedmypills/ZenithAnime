# ZenithAnime
Android-клиент для просмотра аниме

> [!WARNING]
> Приложение не имеет исходников, данный репозиторий нужен только для FAQ и Issues!

Возможности приложения:
- [x] Каталог Shikimori 
- [x] Онлайн-просмотр из разных источников
- [x] Плеер на основе libmpv
- [x] Локальные закладки 
- [x] Облачные закладки
- [x] Поддержка Android TV
- [ ] Интерполяция видеоряда 
- [ ] Апскейл видеоряда 
- [ ] Торрент-стриминг
- [x] Прекэширование следующей серии
- [ ] Загрузка аниме для оффлайн-просмотра
- [ ] Мультиплатформа (Android, Desktop)
